.compileJavaClass(
className, code
)

1.8.4

A Java Development Kit (JDK) must be installed (and potentially used to start Minecraft) in order to compile whole classes.

Compiled classes can't be accessed from any guest language, but must be either stored through FGlobalVars#putObject(java.lang.String,java.lang.Object) or retrieved from this library. Unlike normal hot swapping, already created instances of the class will not be updated. Thus, it's important to know which version of the class you're using when instantiating it.

Parameter Type Description

the fully qualified name of the class, including the package
