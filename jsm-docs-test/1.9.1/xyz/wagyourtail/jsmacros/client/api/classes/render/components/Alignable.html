The alignment must be of the format [left|center|right|x%]On[left|center|right|x%]. The input is case-insensitive. The first alignment is for the element this method is called on and the second is for the other element. As an example, LeftOnCenter would align the left side of this element to the center of the other element.

.alignVertically(
other, alignment, offset
)

1.8.4

The alignment must be of the format [top|center|bottom|y%]On[top|center|bottom|y%]. The input is case-insensitive. The first alignment is for the element this method is called on and the second is for the other element. As an example, BottomOnTop would align the bottom side of this element to the top of the other element. Thus, the element would be placed above the other one.
