.isCursed(
)

1.8.4

Curses are enchantments that can't be removed from the item they were applied to. They usually only have one possible level and can't be upgraded. When combining items with curses on them, they are transferred like any other enchantment. They can't be obtained through enchantment tables, but rather from loot chests, fishing or trading with villagers.

Returns:

boolean

true if this enchantment is a curse, false otherwise.
