.getDemandBonus(
)

1.8.4

The demand bonus is globally applied to all trades of this type for all villagers and players. It is used to increase the price of trades that are in high demand. The demand is only calculated and updated on restock. Note that a villager can always restock, even if no items were traded with him. Updating the demand is done with the following formula: 

 demand = demand + 2 * uses - maxUses

Thus trading only half of the max uses will not increase the demand. The demand is also capped at 0, so it can not decrease the price.

Returns:

int

the demand bonus for this trade.
